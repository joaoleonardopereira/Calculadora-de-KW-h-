# CalculadoraDeKW/H

O programa, feito em **Python**, tem como objetivo **calcular o gasto de KW/h da geladeira do usuário** e **calcular o valor do KW/h em R$.**
Ele pergunta a **marca** da geladeira, a **potencia** e quantas horas fica **ligada** durante o dia.
Fórmulas utilizadas: KW/h = (potencia * horas ligadas * 30) / 1000
R$ = KW/h * 0.70 

# INSTRUÇÕES DE USO

**NÃO utilize letras** nas variáveis numéricas (exemplo: utilizar o W para especificar que são Watts: 250W)
